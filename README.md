![](https://steamuserimages-a.akamaihd.net/ugc/913541295205023522/042E786F1175C84B3374E62B7B20DE0EAC29C286/)

### Installation:
Git clone:

```bash
$ git clone https://github.com/Nerq1337/Orion-SMS-Bomber.git
```

Install the modules with:

```bash
$ npm i
```

### How to start:

```bash
$ node Orion
```
